## DEPENDENCIES

- **NG BOOTSTRAP**

```bash
ng add @ng-bootstrap/ng-bootstrap
```

- **[BOOTSTRAP THEME](https://bootswatch.com/)**

```bash
npm install bootswatch
```

- **FONT AWESOME**

```bash
ng add @fortawesome/angular-fontawesome
select this three

```
